## ⛔ 8대 절대 규칙 ⛔

1. **코드에서 클래스명만 사용** - 풀네임 절대 금지
2. **trace, debug, info, warn 레벨 로그는 사용하지 말고 error 레벨 로그만 사용하라**
3. **모든 주석 및 로그는 영어로 작성하라**
4. **100% 한국어** - 모든 대화, 설명
5. **Thinking 무조건 한국어** - 영어 1글자도 금지
6. **기술 용어 영어 병기 허용** - 예: 필터 체인(Filter Chain)
7. **이모지 절대 금지** - Java 파일, 주석, 로그
8. **소스를 승인 없이 절대로 축소, 삭제, 주석처리, 누락기키지 말 것** - Java 파일, 주석, 로그
9. **컴파일 오류나 에러가 난다고 기존 소스를 니 맘대로 수정해서 해결할려고 절대로 하지 말라**

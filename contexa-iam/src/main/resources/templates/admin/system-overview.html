자율 학습 시스템 건강도 및 전체 시스템 리소스 현황

📊 Grafana 대시보드 설정

시스템 전체 메트릭을 시각화하려면 Prometheus와 Grafana를 실행해야 합니다.

1 Grafana 대시보드 Import

system-overview.json 파일을 Grafana에 import해야 합니다:

파일 위치: D:\projects\contexa\monitoring\grafana\dashboards\system-overview.json

Grafana → Dashboards → New → Import → Upload JSON file

2 대시보드 구성 요소

이 대시보드는 다음 메트릭을 표시합니다:

  • • 시스템 건강도 - 전체 시스템 건강 점수 (0-1)
  • • 활성 인시던트 - 현재 진행 중인 보안 인시던트 수
  • • 위협 레벨 - 실시간 위협 수준 (0-1)
  • • 시스템 리소스 - CPU, 메모리, 디스크 사용률
Grafana 대시보드 열기 Prometheus 열기

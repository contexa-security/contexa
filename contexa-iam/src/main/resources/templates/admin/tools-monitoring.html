도구 실행 모니터링 (MCP + SOAR)

MCP 도구 및 SOAR 도구 실행 현황 통합 대시보드

📊 Grafana 대시보드 설정

MCP 및 SOAR 도구 실행 메트릭을 시각화하려면 Prometheus와 Grafana를 실행해야 합니다.

1 Grafana 대시보드 Import

tools-monitoring.json 파일을 Grafana에 import해야 합니다:

파일 위치: D:\projects\contexa\monitoring\grafana\dashboards\tools-monitoring.json

Grafana → Dashboards → New → Import → Upload JSON file

2 대시보드 구성 요소

이 대시보드는 다음 메트릭을 표시합니다:

  • • MCP 도구 실행 - 성공/실패 횟수, 평균 실행 시간, 캐시 히트율
  • • SOAR 도구 실행 - 승인/거부 비율, 필터링 현황, 실행 성능
  • • 도구별 상세 분석 - 도구별 성공률, 평균 실행 시간
  • • 실시간 현황 - 초당 도구 실행 횟수, 에러율
Grafana 대시보드 열기 Prometheus 열기

핵심 메트릭 개요

MCP 도구 실행
mcp.tool.executions
성공/실패 추적
SOAR 도구 승인율
soar.tool.approval.rate
승인/거부 비율
MCP 캐시 히트율
mcp.cache.hit.rate
성능 최적화 지표
평균 실행 시간
tool.execution.time
밀리초 단위

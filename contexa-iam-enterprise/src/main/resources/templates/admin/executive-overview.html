📊 Grafana 대시보드 설정

임원진용 통합 보안 건강도 메트릭을 시각화하려면 Prometheus와 Grafana를 실행해야 합니다.

1 Grafana 대시보드 Import

executive-overview.json 파일을 Grafana에 import해야 합니다:

파일 위치: D:\projects\contexa\grafana\dashboards\executive-overview.json

Grafana → Dashboards → New → Import → Upload JSON file

2 대시보드 구성 요소

이 대시보드는 다음 메트릭을 표시합니다:

  • • 전체 보안 건강도 - ZeroTrust(30%) + Evolution(30%) + Vector(20%) + HCAD(15%)
  • • 도메인별 건강도 추이 - 각 도메인의 시계열 변화
  • • 보안 이벤트 처리량 - 5분 단위 이벤트 처리 현황
  • • 종단간 처리 지연시간 - P50, P95, P99 백분위수

감사 준비 완료

이 대시보드는 감사 목적으로 설계되었습니다. 모든 메트릭은 Prometheus에 영구 저장되며 추적 가능합니다.

VectorStore 모니터링

contexa PgVector 실시간 메트릭 대시보드 (Prometheus + Grafana)

📊 Grafana 대시보드 설정

VectorStore 메트릭을 시각화하려면 Prometheus와 Grafana를 실행해야 합니다.

1 Docker Compose 실행

프로젝트 루트에서 다음 명령어를 실행하여 Prometheus와 Grafana를 시작합니다:

docker-compose -f docker-compose-monitoring.yml up -d
2 Grafana 접속 확인

Grafana가 실행되면 아래 링크로 접속할 수 있습니다:

http://localhost:3000

초기 로그인: admin / contexa

3 대시보드 자동 로드 확인

"contexa VectorStore 모니터링" 대시보드가 자동으로 프로비저닝됩니다.
Dashboards → Browse → contexa VectorStore 모니터링

중요: Spring Boot 애플리케이션 실행 필요

aicore 모듈이 실행 중이어야 Prometheus가 메트릭을 수집할 수 있습니다.

./gradlew.bat :aicore:bootRun
Grafana 대시보드 열기 Prometheus 열기

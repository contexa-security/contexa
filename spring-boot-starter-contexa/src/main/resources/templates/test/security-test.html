  • 정상 사용자와 계정탈취자가 같은 인증 세션 이후에 어떻게 다르게 감지되는지 증명합니다.
  • 브릿지, 컨텍스트, 프롬프트, LLM 판정, 다음 요청 제어, SaaS evidence가 끊김 없이 이어지는지 검증합니다.
  • UI와 서버가 같은 requestId, sessionId, auth metadata를 보고 있는지 즉시 검증합니다.

// THE COMING CRISIS

AI Agent는 인증을 통과한 뒤,
차원이 다른 위협이 됩니다.

국내외 대형 침해는 사람이 인증을 통과한 뒤 발생한 것입니다.
AI Agent가 인증을 통과하면, 피해의 규모와 속도가 근본적으로 달라집니다.

첫 번째 차이
패턴 불규칙
사람은 인증 이후에도 습관이 있습니다.
같은 시간, 같은 경로, 같은 방식.
AI Agent는 상황에 따라 행동 패턴이 빠르게 달라질 수 있어
기존 탐지 체계로는 식별이 어렵습니다.
두 번째 차이
24시간 공격
사람은 인증 이후에도 쉬어야 합니다.
AI Agent는 인증 한 번으로 24시간 내부에서 활동합니다.
새벽에도, 휴일에도 멈추지 않습니다.
세 번째 차이
진위 불가
인증을 통과한 정상 Agent와 동일한 자격증명.
동일한 권한, 동일한 행동 패턴.
고정 규칙만으로는
정상과 위협을 안정적으로 구분하기 어렵습니다

인증 이후의 행동은 불규칙하고, 24시간 멈추지 않으며, 진위 구별도 되지 않습니다.
인증 이후를 방어하지 못하는 보안 체계는 AI Agent 시대에 구조적 한계를 드러냅니다.

// HOW CONTEXA WORKS

인증 이후, Contexa는 이렇게 판별합니다.

규칙은 "이 행동이 규칙에 위반하는가"를 봅니다. 규칙에 없으면 통과입니다.
Contexa는 "이 행동이 이 사람의 평소와 같은가"를 봅니다.
규칙에 없어도, 평소와 다르면 찾아냅니다.

01 — 학습
정상을 정의한다
각 사용자의 접근 시간, 접근 대상,
업무 환경, 행동 흐름을 AI가 학습합니다.
미리 정의된 규칙이 아닌
개인별 행동 프로파일입니다.
02 — 비교
평소와 다른가
매 순간 현재 행동을 프로파일과 비교합니다.
평소 9시에 출근하는 사람이 새벽 3시에 접속.
한 번도 열지 않던 핵심 DB에 접근.
규칙에 없어도 AI가 감지합니다.
03 — 대응
AI가 스스로 대응
이상이 감지되면 사람의 판단을 기다리지 않습니다.
세션 제한, 권한 축소, 접근 차단.
AI가 자율적으로 대응하는 구조입니다.
